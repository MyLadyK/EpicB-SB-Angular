Ayuda y Modo Demo

¿Qué es el modo demo?

Cuando el backend (API o base de datos) no está disponible, la aplicación entra en Modo Demo y muestra datos simulados. Esto permite probar el flujo de usuario, visualizar batallas y el ranking aunque el servidor real esté caído.

¿Cómo funciona el modo demo?

  • Las batallas y el ranking mostrarán datos de ejemplo.
  • No se guardarán cambios reales ni se modificará la base de datos.
  • Puedes navegar por toda la app y ver cómo funciona.

¿Cómo probar la app?

  • Selecciona usuarios y personajes para lanzar una batalla.
  • Consulta el ranking para ver posiciones simuladas.
  • Si ves el banner de Modo Demo, significa que los datos son de ejemplo.

¿Qué pasa cuando el backend vuelve a estar disponible?

Cuando el backend vuelva a estar disponible, la app volverá a mostrar datos reales automáticamente.

¿Dónde puedo obtener soporte?

Si tienes dudas o necesitas soporte, contacta con el administrador del sistema.
